# DevopsCW
Classwork repository for Devops SWDV 460



Project Overview:
Project is a C application that reads an XML file provided on the command line, parses the file using the libxml2 library, and prints all book titles to the console.

Created for Linux/Red Hat EC2 assignment required: 
  1. Reading an xml file from the command line
  2. Parsing xml using xml library
  3. Displaying all book titles from the xml file
  4. Creating a Makefile for building the application
